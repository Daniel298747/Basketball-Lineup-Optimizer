# Basketball-Lineup-Optimizer

## About
As someone who grew up playing and watching basketball, I wanted to combine my passion 
for the sport with my software development skills. This project was built to solve a 
real problem — how do you pick the best possible lineup from a squad of players?

## What it does
- Takes a roster of players with individual stats and ratings
- Uses algorithmic logic in Java to generate and evaluate optimal lineups
- Displays the results through a browser-based dashboard for easy visualization

## Tech Stack
- **Java** - Core logic and optimization algorithm
- **HTML/CSS/JavaScript** - Browser-based dashboard

## What I learned
- Designing and implementing algorithms in Java
- Connecting backend logic to a frontend dashboard
- Breaking down a real-world problem into a software solution
